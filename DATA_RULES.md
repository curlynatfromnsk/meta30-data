# META30 Data Rules

## Правила именования

| Что | Правильно | Неправильно |
|-----|-----------|------------|
| Хлеб-буханка | Кето хлеб | Льняной кето хлеб |
| Хлеб 90 сек | Кето хлеб за 90 секунд | keto-bread-90 |
| Масло для жарки | Оливковое масло / Масло авокадо | Масло виноградной косточки |
| Углеводы | чистые углеводы | net carbs |
| Полдник | Полдник | Снэк |
| Кето-кофе | Кето-кофе | Бронекофе |

## rk ключи (recipe keys)

Ключ в ингредиенте `"rk"` ссылается на рецепт в `app.js`. Допустимые значения:

```
tortilla, keto-bread, keto-bread-90, rostbif, guacamole,
mayo, broccoli-slaw, zucchini-waffles, keto-buns, keto-muffins,
italian-dressing, ranch-dressing, caul-broccoli, mashed-cauliflower,
keto-latte, broccoli-soup, chicken-drumsticks, egg-bites
```

**Если rk нет в этом списке — ссылка не будет кликабельной.**

Покупные соусы (Соус песто, Релиш) НЕ имеют rk — это магазинные продукты.

## Формат данных

### data_XXXX.json

```json
{
  "menu": [{
    "d": 1,
    "ml": [{
      "n": "Название блюда",
      "t": "07:00",
      "s": "keto",          // keto | breakfast | lunch | snack | dinner
      "m": {"kcal":251, "p":0, "f":28, "nc":0},
      "i": [{"n":"Гхи", "a":"1 ст. л.", "rk":"..."}],
      "r": "Инструкция..."
    }],
    "tt": {"kcal":1912, "p":62, "f":174, "nc":11}
  }],
  "shop": {
    "Неделя 1 (дни 1–7)": {
      "Категория": [{"product":"Продукт", "qty":"100 г"}]
    }
  },
  "prep": {
    "Неделя 1 (дни 1–7)": [
      {"recipe":"Кето хлеб", "ingredient":"Кето хлеб", "qty":"6 порций", "portions":6, "cook_times":"1"}
    ]
  }
}
```

### Формат prep

**data_1800 и data_2000**: одна строка = один рецепт (ingredient = recipe name, qty = порции)

**data_1600**: одна строка = один ингредиент рецепта (ingredient = "Миндальная мука", qty = "42 г")

### articles.json

```json
[{
  "id": "art-1-1",
  "title": "Название",
  "level": 1,
  "levelName": "Почему это работает",
  "body": "<p>HTML контент...</p>",
  "readingTime": 5
}]
```

## Ограничения

| Параметр | Значение |
|----------|---------|
| kcal | Диапазоны: 1600–1800, 1800–2000, 2000–2200 |
| Белок | ≤ 80 г/день |
| Чистые углеводы | ≤ 25 г/день |
| Слотов в дне | 5 (keto, breakfast, lunch, snack, dinner) |
| Дней | 30 |
| Повтор блюда | Не чаще чем раз в 3 дня в том же слоте (кроме кето-кофе) |

## Категории shop

Масла и жиры, Молочные и яйца, Мясо и рыба, Овощи и зелень, Орехи и семена, Мука и выпечка, Напитки, Приправы и соусы, Прочее

## Что НЕ должно быть в prep

Покупные продукты: Релиш, Соус песто — они идут в shop → Приправы и соусы.

## Валидация

```bash
# Из корня meta30-data (репозитории рядом):
python3 ../meta30_cabinet/scripts/validate_data.py .
```

Скрипт проверяет: структуру, макросы, rk ключи, запрещённые строки, дубли в prep, shop полноту, яйца в штуках.

## Скрипты (хранятся в meta30_cabinet/scripts/)

| Скрипт | Назначение |
|--------|-----------|
| `validate_data.py` | Автотесты данных (запускать перед каждым пушем) |
| `parse_menu_csv.py` | CSV из Notion → JSON (menu + встроенный валидатор) |
| `recalc_shop_prep.py` | Пересчёт shop/prep из menu |
| `build_articles.py` | Markdown из Notion → HTML для articles.json |
